# NYC Airbnb Market Analysis & Power BI Dashboard

## Project Overview
This project analyzes the **New York City Airbnb Open Data** to understand pricing patterns, neighborhood dynamics, room-type performance, seasonal availability, and host participation. The dataset covers thousands of listings across all five NYC boroughs.

An interactive **Power BI dashboard** was developed to convert raw listing data into clear, actionable insights through structured analysis and visual storytelling.

## Objectives
- Examine price distribution across boroughs and neighborhoods  
- Identify high-density and high-demand areas  
- Compare performance of different room types  
- Analyze seasonal availability and demand patterns  
- Provide market-level insights using interactive visuals  

## Dataset
The dataset used is the **New York City Airbnb Open Data** sourced from Kaggle. It contains detailed listing information including prices, room types, availability, reviews, host details, and neighborhood-level geography.

## Key Insights
- Airbnb listings are highly concentrated in Manhattan and Brooklyn  
- Most listings fall within mid-range price segments  
- Availability varies significantly across seasons  
- Entire homes command higher prices compared to other room types  
- The market is largely driven by individual hosts
